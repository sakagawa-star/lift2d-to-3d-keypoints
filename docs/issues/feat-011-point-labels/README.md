# feat-011: visualize_points_2d.py 基準点番号ラベル表示オプション

## Status

Closed（2026-06-07 実機確認OK）

## 概要

`visualize_points_2d.py` で静止画にプロットする基準点の横に、基準点の番号（ObjectName から抽出した数字部分のみ）を描画するオプション機能を追加する。

## 背景

現状は赤い円と黄色い中心点のみが描画され、どの点がどの基準点（基準_01 など）に対応するか画像上で判別できない。番号ラベルがあれば、2D座標の入力ミス（点の取り違え）を目視で発見しやすくなる。

## 要求元

ユーザー依頼（2026-06-07）。ラベルは「番号のみ」（例: `01`）と確認済み。OpenCV putText は日本語を描画できないため、ObjectName 全体（「基準_01」）は表示しない。
