# feat-004: gsplatバッチレンダリングパイプライン

## Status: Closed

## 概要

Blenderで人のキーポイント（頭部・眼球）に追従するカメラのアニメーションを、3D Gaussian Splatting（3DGS）シーン上でバッチレンダリングするパイプラインを実装する。

KIRI Engine Blenderアドオンで3DGSシーンのカメラ映像をレンダリングしているが、CPUボトルネックにより遅い。これをgsplat（CUDA）で置き換える。

## 成果物

- `phase4/` ディレクトリに配置するgsplatバッチレンダリングスクリプト
- PLYファイルとカメラポーズJSONを受け取り、フレームごとに画像を出力

## 開発ステップ

1. **Step 1: PLYロードと基本確認** — PLYファイルを読み込み、各属性のshape・値域を確認
2. **Step 2: 固定カメラで1枚レンダリング** — ハードコードした仮カメラポーズで1枚レンダリング
3. **Step 3: カメラポーズJSONの読み込み** — 全フレームの連番PNGを出力
